=head1 NAME

Perl6::Overview::Reduce - Reduce metaoperator

=head1 DESCRIPTION

Uses of the reduction meta-operator:

    [+]                    # sum(...)
    [*]                    # product
    [*] 1..$x              # factorial
    [~]                    # join('', ...)
    [+^]                   # numeric parity
    [~^]                   # string parity
    [?^]                   # boolean parity
    [<]                    # monotony
    [==]                   # equality
    [//]                   # first defined element
    [||]                   # first true element
    [=>]                   # linked list construction
    [&]                    # all(...)
    [|]                    # any(...)
    [^]                    # one(...)
    [Y]                    # zip(...)
    [,]                    # noop, returns input list
    [;]                    # [[;] 1,2,3] === [1;2;3] == [[1],[2],[3]]
    [>>op<<]               # perform op on each colum
